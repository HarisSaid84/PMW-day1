# Haris Said — Personal Portfolio

A single-page personal portfolio website built with plain HTML, CSS, and JavaScript — no frameworks or build tools required.

## PMW Track

**AI Based 3D Scene Reconstruction**

## What I built today

A polished, responsive personal portfolio site (`index.html`) featuring:

- A hero section with a profile photo, name, and role
- An "Areas of interest" section covering AI, machine learning, and data structures & algorithms
- A technical skills overview
- Education and academic awards
- Two featured projects: **The Iron Nexus** (railway management system in C++) and **CityMind** (AI-driven urban intelligence simulation)
- Leadership and volunteering highlights
- A contact section with email, phone, and LinkedIn

The site uses a terminal/directory-inspired design theme, is fully responsive down to mobile, and respects reduced-motion preferences for accessibility.

## How to open the portfolio

1. Make sure `index.html` and `profile.jpg` are in the same folder.
2. Double-click `index.html`, or open it directly in any web browser.

No installation, server, or internet connection is required (aside from loading the Google Fonts used for typography).
